# 🎓 2027 전문대 수시 합격진단 (매칭 엔진)

내신 등급을 입력하면 전국 **125개 전문대학 · 8,561개 학과·전형**을
**안정 · 적정 · 소신 · 도전** 4단계로 자동 분류해 주는 상담 도구입니다.
단일 HTML 파일이라 인터넷·엑셀 없이 오프라인으로도 동작합니다.

## 🔗 바로 사용
👉 https://hong-jame.github.io/jeonmundae-counseling/

## 주요 기능
- 내신 등급 입력(숫자·슬라이더) → 실시간 합격 가능성 진단
- 판정 근거: **2026학년도 최종등록자 평균등급**
- 필터: 계열 · 권역 · 학제(2/3/4년제) · 대학/학과 검색 · 수능최저 학과만 · 입결없는 학과 숨기기
- 정렬: 적합도순 / 합격선 낮은순 / 경쟁률 낮은순 / 취업률 높은순
- 학과 카드: 평균·최저등급, 3년 등급 추이, 경쟁률, 취업률, 충원율, 모집인원, 수능최저 여부

## 판정 기준
`차이 = 최종등록자 평균등급 − 내 등급`

| 판정 | 기준 |
|---|---|
| 🟢 안정 | 1.0등급 이상 여유 |
| 🔵 적정 | 0.2~1.0 여유 |
| 🟡 소신 | ±0.2 내외 |
| 🟠 도전 | 0.2~1.2 부족 |
| 🔴 어려움 | 1.2등급 초과 부족 |

## 파일 구성
- `index.html` — 배포용 완성 파일(데이터 내장)
- `_template.html` — 데이터 자리(`/*__DATA__*/`) placeholder 템플릿
- `_data.json` — 입결 시트에서 추출한 컴팩트 데이터(17필드 × 8,561행)
- 재빌드: `_data.json` 내용을 `_template.html`의 placeholder에 치환
